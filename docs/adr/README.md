# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the Gravtory project.

ADRs document significant architectural decisions, their context, and rationale.
We use the format described by [Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).
